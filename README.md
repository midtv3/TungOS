# TungOS
TungOS is an operating system based on Ubuntu Cinnamon. It is themed around the character Tung Tung Tung Sahur. It is currently closed source because I am not sure how to open source what I built.

# Features

 - Preinstalled Discord, Spotify, and Steam
 - Tung Tung Tung Sahur custom wallpaper
 - Custom Neofetch

# Upcoming Features
These features are planned for a future release of TungOS. 
 - Debloated, No Snap or Canonical Spyware
 - Custom Boot Screen
 - Modified GRUB Titles
 - Smaller image size

# Feature Suggestions
